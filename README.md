# Personal Portfolio Website

A personal portfolio website showcasing my projects, skills, and background as a web development student and programmer.

## Description

This website was created to serve as a professional homepage where visitors can learn more about me, explore my web development projects, and view the skills and technologies I have worked with. The site highlights my coursework, coding experience, and project portfolio in a clean, organized, and responsive design. The target audience includes instructors, classmates, potential employers, and anyone interested in my work.

## Features

- Responsive design for desktop, tablet, and mobile devices
- Homepage with personal introduction and background information
- Portfolio/projects section showcasing completed work
- Navigation menu for easy browsing
- Clean and modern user interface
- Styled layouts using CSS Flexbox and responsive techniques
- Interactive webpage elements

## Technologies Used

- **HTML5**: Used to structure the content and create semantic webpage elements.
- **CSS3**: Used for styling, layouts, colors, responsiveness, and visual enhancements.
- **JavaScript**: Used to add interactivity and dynamic functionality.
- **GitHub Pages**: Used to deploy and host the website online.
